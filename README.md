# Iporá: Vias-Inteligentes

## Descrição
O presente projeto tem como objetivo resolver um dos problemas percebidos na cidade de Iporá durante o conteúdo de Pensamento Computacional, que é o problema da falta de infraestrutura de trânsito, ou seja, o projeto tem como público alvo todos que utilizam o trânsito de Iporá para se locomoverem.
