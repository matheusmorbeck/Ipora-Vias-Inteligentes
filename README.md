# Ipor-Vias-Inteligentes
Projeto desenvolvido no curso de Ciência da Computação no IF Goiano. O mesmo foi pensado na matéria de Introdução ao Ambiente Universitário e Computação, principalmente no conteúdo de Pensamento Computacional, visto que, o foco do projeto é resolver algum problema percebido na cidade de Iporá, com isso, escolhemos os problemas de trânsito.
